## 조건문

### if 조건문

- 조건이 True인 경우 이후 들여쓰기 되어 있는 코드 블록을 실행
- 그 외의 경우 else 이후 들여쓰기 되어 있는 코드를 실행
  - else의 경우 조건을 선택적으로 부여할 수 있음

```python
a = 5
if a > 5:   # if의 조건을 충족하지 않음
    print('5 초과')
else:   # 이하 들여쓰기 된 코드 실행
    print('5 이하')
```

- 조건문 실습 문제 : 홀수/짝수 구분하기
  
```python
num = int(input())
if num % 2 == 1:
    print('odd')
else:
    print('even')
```

- 복수의 조건식을 사용할 경우 'elif' 사용

### 중첩 조건문

- 조건문을 다른 조건문과 중첩하여 사용할 수 있음(들여쓰기 주의)

```python
hat = 'red'
clothes = 'blue'
if hat == 'red':
    if clothes == 'red':
        print('red')
    else:
        print('semi red')
else:
    print('not red')
```

### 조건 표현식

- 조건 표현식을 일반적으로 조건에 따라 값을 정할 때 사용
- 삼항 연산자라고도 부름
  
```python
value = num if num >= 0 else value = 0
```

- 위 조건 표현식은 아래 조건문과 동일한 작업 수행

```python
if num >= 0:
    value = num
else:
    value = 0
```

### 반복문

- while
  - 조건식이 참인 경우 들여쓰기 된 코드를 반복적으로 코드 실행
    - PEP8에서는 들여쓰기 여백으로 4space 권장 
  - 종료 조건에 해당하는 코드로 반복문을 종료시켜야 함
    - 종료 조건을 지정하지 않으면 무한 반복될 수 있음

- for
  - 반복 가능한 개체를 모두 순회하면 종료됨
  - 순회 가능한 모든 개체(iterable)의 요소를 모두 실행
  - 별도의 종료 조건이 필요하지 않음
    - iterable : 순회할 수 있는 자료형과 함수  
    - 예) string, list, dict, tuple, range, set, enumerate

- for enumerate 순회
  - enumerate(리스트, start = )
  - <u>start에 숫자를 넣으면 몇 번째 인덱스부터 시작할지 설정 가능</u>

- **List Comprehension**
  - 표현식과 제어문을 통해 특정 조건의 리스트를 간결하게 생성 가능
    
    ```python
    cubic_list= [number ** 3 for number in range(1,4)]
    ```
    
    ```python
    cubic_dict = {number: number ** 3 for number in range(1,4)}
    ```

- 반복 제어
    - break : 반복문 즉시 종료(반복문 탈출)
    - continue : continue 이후의 코드 블록을 수행하지 않고, 다음 블록(조건) 실행
    - for-else : 끝까지 반복문을 실행한 후 else문 실행
      - break를 통해 중간에 종료되는 경우 else 이하는 실행되지 않음
      - for 구문이 잘 실행되는지 체크하는 용도로도 사용됨
      - for와 else를 동일한 위치에 들여쓰기 하면 됨
    ```python
    for char in 'apple':
        if char == 'b':
            print('b')
            break
    else:
        print('b가 없습니다')
    ```
    - pass : 아무 동작 없음(문법적으로 필요한 경우에만 삽입)
