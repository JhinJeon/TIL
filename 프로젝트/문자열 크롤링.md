# 한국어 크롤링

re 라이브러리를 이용해서 ㄱ-ㅎ, ㅏ-ㅣ, 가-힣 을 기준으로 컴파일하면 한국어만 크롤링 할 수 있다.

```python
import re
text = "안녕하세요 ㅎㅎ 반갑습니다"

re.complie('[가-힣]+').findall(text)  # [안녕하세요, 반갑습니다]

# 아래는 정규식([ㄱ-ㅎ, ㅏ-ㅣ])에서 일치하는 부분을 제외하고 저장
re.compile('[ㄱ-ㅎ|ㅏ-ㅣ]+').sub('',text)
```
