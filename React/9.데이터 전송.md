# 데이터 삽입하기

- 데이터를 태그 안에 넣으려면 중괄호로 감싸서 객체로 변환해야 한다.

```js
const title = ['이름', '전공']

return(
    <table>
        <tbody>
            <tr>
                <th>이름: {title[0]}</th>
            </tr>
            <tr>
                <th>전공: {title[1]}</th>
            </tr>
        </tbody>
    </table>
)
```
