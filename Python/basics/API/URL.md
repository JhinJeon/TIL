## API

- Application Progamming Interface
- 앱이 사용자의 요청에 응답하는 접점(인터페이스)

모든 프로그램은 요청과 응답으로 구성됨

- get : 데이터를 조회할 때
- post : 데이터를 수정할 때(댓글 수정 등)

## URL의 규칙

- https:// : 통신 프로토콜 규약
- www.blabla.com/ : 기본(Base) URL
- path : 기본 URL의 세부 경로(메뉴)
- ?params : parameter, '&' 로 파라미터 간 구분
  - key, value 일대일 대응
