
## set에서 에러 내지 않고 값 지우기

- .discard() 이용

```python
my_set = {'민트', '초코', '피자'}
my_set.discard('민트')
print(my_set)  # {'초코','피자'}
```

- 주의 : tuple, dict, list에는 사용할 수 없음

## 딕셔너리 연습 2


### .get()으로 없는 키를 찾을 때도 에러가 나지 않도록 설정

```python
my_dict = {'민트': '초코',
           '피자': '파인애플'}

print(my_dict.get('탕수육', '없는데요?'))
# 결과 : 없는데요?
```

### .update()로 딕셔너리 값 수정

- Type A

```python
my_dict = {'민트': '초코',
           '피자': '파인애플'}
my_dict.update(민트='만먹음', 피자='파인애플안넣어')
print(my_dict)
# 결과 : {'민트': '만먹음', '피자': '파인애플안넣어'}
```

- Type B

```python
my_dict = {'민트': '초코',
           '피자': '파인애플'}
my_dict.update({'민트': '만먹음', '피자': '파인애플안넣어'})
print(my_dict)
```
